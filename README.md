# amazon-tracker

amazon-tracker is a little tool to track and notify when Amazon prices are down.

## Installation

Use git. `sudo apt install git`

```bash
git clone https://github.com/ultrajacobboy/amazon-tracker
```

## Features  
Evades bot detection with fake user agent and proxies
Notifies on Discord when you want  
Easy to use

## Usage

``` 
python3 main.py -a  
```
Add an item to track

```
python3 main.py -d
```
Delete an item
```
python3 main.py -r
```
Begin tracking
